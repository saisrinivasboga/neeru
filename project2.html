Project 1 :  Rock-Papers-scissors

↪ Project Description

This is an interactive Rock, Paper, Scissors game built using Python's tkinter library. The game features a graphical user interface (GUI) where users can choose between "Rock", "Paper", or "Scissors" by clicking buttons. The computer also makes a random choice, and the result is displayed on the screen. The game keeps track of the number of wins for both the user and the computer and displays the updated score after each round.

In the game, players choose one of three options: rock, paper, or scissors, and compete against a computer that randomly generates its choice. The winner is decided based on these rules:

  • Rock beats Scissors.
  • Scissors beats Paper.
  • Paper beats Rock.

↪ Programming Concepts Used

  • Use of Python's random module for the computer's choice.
  • Conditional statements (if-elif-else) to determine the game's logic.
  • Scorekeeping over multiple rounds.
  • The game uses the tkinter library to create a GUI. This includes widgets such as Label, Button, and Entry to interact with the user and display results.
  • Lambda functions are used in the button commands, such as command=lambda: play_game("rock"), to pass arguments to the play_game() function when a button is clicked.
  • The program continuously waits for user input through the root.mainloop() function. This is part of the event loop in tkinter, which listens for events like button clicks and handles them accordingly.

↪ Sample Workflow

  1. The user clicks one of the buttons ("Rock", "Paper", or "Scissors").
  2. The computer randomly selects its move.
  3. The result is displayed on the screen, showing whether the user won, lost, or tied the round.
  4. The score is updated and displayed, showing the number of wins for both the user and the computer.

Project 2 :  Password Generator

↪ Project Description

The Password Generator is a simple Python-based GUI application built using the tkinter library. It allows users to generate random passwords and copy them to the clipboard for easy use. The generated passwords are composed of a mix of uppercase letters, lowercase letters, numbers, and special characters, providing a variety of combinations suitable for secure password creation.

  • GUI Interface: A user-friendly graphical interface built with tkinter includes a label, button, and entry widget to display the generated password.
  • Clipboard Functionality: The password can be easily copied to the clipboard using the pyperclip library for quick pasting into other applications.

↪ Programming Concepts Used

  1. Modules and Libraries
    • tkinter: Used to create the graphical user interface (GUI).
    • random: Used to generate random characters for the password.
    • pyperclip: Used to copy the generated password to the clipboard.
  2. Functions
    • generate(): A function that generates a random password by selecting random characters from a predefined list.
    • copyclipboard(): A function that copies the generated password to the clipboard using the pyperclip module.
  3. Graphical User Interface (GUI) Design
    • Label: Used to display static text in the GUI.
    • Button: Used to create interactive buttons for generating a password and copying it to the clipboard.
    • Entry: Used to display the generated password in a text box.
    • pack(): Used to position and manage widgets in the GUI.
  4. Dynamic Updates
    • The value of passwrd is updated using set() and displayed in the Entry widget dynamically.
  5. GUI Window Management
    • Tk(): Initializes the root window.
    • geometry(): Sets the dimensions of the window.
    • mainloop(): Starts the event loop to keep the GUI active and responsive.

↪ Sample Workflow

  1. User clicks the "Generate Password" button.
  2. A random password is generated and displayed in the text entry field.
  3. User clicks the "Copy to Clipboard" button to copy the generated password
