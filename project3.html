Project 1 :   Calculator

↪ Project Description

The Web-Based Calculator project is a user-friendly application designed to perform various mathematical calculations directly from a web browser. It provides a seamless and interactive interface, making it suitable for both basic and advanced mathematical operations. The calculator is built with responsive design principles, ensuring compatibility across different devices such as desktops, tablets, and smartphones.

↪ Features

  1. Basic Arithmetic Operations:
    • Perform addition, subtraction, multiplication, and division.
    • Easy-to-use interface for straightforward calculations.
  2. Advanced Mathematical Functions:
    • Square root (√), power (^), and logarithmic functions (log).
    • Trigonometric functions including sine (sin), cosine (cos), and tangent (tan) with angle inputs in degrees.
    • Error handling for undefined results, such as tan(90)
  3. Error Handling:
    • Real-time alerts for invalid inputs or calculations (e.g., division by zero or undefined tangent angles).
  4. History Log:
    • Displays a scrollable history of previous calculations.
    • Provides an easy reference for past results.
  5. Backspace Functionality:
    • Allows users to delete the last entered character, offering precise control over input.
  6. Dynamic Interface:
    • Blue for numeric keys.
    • Dark blue for special functions like trigonometric and logarithmic calculations.
    • Light gray for clear (C) and equals (=) buttons.
  7. Customizable Themes:
    • Light and dark mode toggle to suit different lighting conditions.
    • Aesthetic appearance with smooth transitions.

↪ Usage Scenarios

  • Students and professionals performing calculations for studies or work.
  • Engineers and mathematicians needing advanced functions like logarithms or trigonometric computations.
  • General users needing a reliable, accessible calculator for everyday use.
